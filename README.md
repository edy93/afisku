# Laravel 8 RESTful APIs Boilerplate
Boilerplate RESTful APIs menggunakan Laravel 8

## Fitur
- *Auth* (*register, verify, reset password, confirm password*)
-  API with`Laravel/Sanctum`

## *Requirements*
- PHP >= 7.3
- Git
- Composer

## *Installing*
- *Clone* repo:
```
git clone 
```
- Jalankan:
```
$ composer install
```
- *Setup* file `.env` (Jika tidak ada).
